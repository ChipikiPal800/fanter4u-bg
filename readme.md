what is this?

fanteros is not just another unblocked games site. it's a full desktop operating system that runs in your browser. you get a taskbar, a start menu, draggable windows, widgets, wallpapers, sound effects, right click menus, and everything else you would expect from a real os. but instead of microsoft word and excel, you get thousands of games, a working proxy, a chat app, and tools that help you actually play at school without getting caught.

you don't install anything. you can  most likely just open the link and it works.

features list ok?

there's a full desktop environment with windows you can drag and resize. the taskbar shows your running apps and has a start menu with categories. you can pin apps to your desktop and the positions save automatically.

the game hub has over five thousand roms across ten consoles. we also have webports of modern indie games like hollow knight, inscryption, celeste, cuphead, buckshot roulette, and more are being added all the time.
